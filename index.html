Portfolio

Elder Music Project

Talent Showcased: Server-side functionality, client-side functionality, responsive design

Objective: Teach the user how to improvise music on guitar

Target Audience: Teenagers and young adults

Highlights:

  • On the homepage, the user can load a random music track based on their selection of a music key, interval, and tempo from a form.
  • AJAX calls are used with JavaScript and jQuery to dynamically load content on the website in order to decrease loading times and to make page transitions clean.
  • Illustrated music chords and scales are loaded onto a generated page based off the user’s selection. The chords can be clicked on to display variations of the chord.
  • Descriptions, tooltips, and an instructional video are featured to guide users through each page and show them how each feature works.
  • Over 180 random tracks are loaded from an external storage source to save file space on the website.
  • Responsive design allows page elements to reposition proportionally based on what device the user has.

Marc Music Market

Talent Showcased: Server-side functionality, database management, responsive design

Objective: Inform the user about instruments for sale

Target Audience: Teenagers and adults

Highlights:

  • Information about each product is stored within a MySQL database and is transferred over to the webpage using PHP.
  • When logged in as an administrator, the user can add and remove products, as well as store locations. To try it, log in with the username of “admin” and password of “admin”.
  • An interactive panel on the top of the products page allows the user to be more selective about what type of products the page displays.
